Introducción

Este proyecto detalla la implementación de consultas avanzadas y la creación de objetos tipo VIEW en el sistema de gestión de la Clínica TodoSalud. El enfoque principal es el dominio de la integridad referencial y la consolidación de información mediante uniones (JOINS) y subconsultas en Oracle SQL Developer.

Desarrollo y Consultas SQL

1. Consolidación de Información Médica

Paso 5.1: Creación de vista para médicos y sus especialidades.

CREATE OR REPLACE VIEW v_medicos_especialidades AS
SELECT m.codm, m.tipodocm, m.nrodocm, m.paternom, m.maternom, m.nombresm, m.celularm, m.fechanac, m.g, e.nombreesp
FROM tmedico m
JOIN tespecialidaddelmedico em ON m.codm = em.codm
JOIN tespecialidad e ON em.codesp = e.codesp;

-- Visualización de la vista
SELECT * FROM v_medicos_especialidades;

2. Conteo y Filtrado de Especialización

Paso 5.2 y 5.3: Identificación de médicos con múltiples especialidades.

-- Médicos con 2 o más especialidades
SELECT nombresm, paternom, maternom, COUNT(*) AS cantidad_especialidades
FROM v_medicos_especialidades
GROUP BY nombresm, paternom, maternom
HAVING COUNT(*) >= 2;

3. Identificación de Brechas (Registros Huérfanos)

Paso 5.4: Detección de enfermedades que nunca han sido diagnosticadas.

CREATE OR REPLACE VIEW v_enfermedades_no_detectadas AS
SELECT * FROM tenfermedad
WHERE code NOT IN (SELECT code FROM tenfermedadeneldiagnostico);

SELECT * FROM v_enfermedades_no_detectadas;

4. Análisis de Especialidades sin Cobertura

Paso 5.5 y 5.6: Especialidades que no cuentan con médicos asignados.

CREATE OR REPLACE VIEW v_especialidades_sin_medico AS
SELECT * FROM tespecialidad
WHERE codesp NOT IN (SELECT codesp FROM tespecialidaddelmedico);

-- Cuantificar el déficit
SELECT COUNT(*) AS total_sin_medico FROM v_especialidades_sin_medico;

5. Auditoría de Calidad: Diagnósticos Incompletos

Paso 5.7: Pacientes que tienen un diagnóstico pero sin enfermedad asociada.

CREATE OR REPLACE VIEW v_pacientes_sin_enfermedad AS
SELECT p.nombresp, p.paternop, p.maternop
FROM tpaciente p
JOIN tdiagnostico d ON p.codp = d.codp
WHERE d.codd NOT IN (SELECT codd FROM tenfermedadeneldiagnostico);
